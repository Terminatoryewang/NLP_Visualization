Welcome to the NLP Visualization Demo

This demo is created to show the NLP visualization results of the database. Some functions are not completed. Clicking them will return to the homepage.

The demo will mainly include four parts: overall visualization of the selectable range, interactive visualization focusing on a certain article, word cloud, and AI part.

The AI ​​part will include two important contents: using the new data added in the previous data cleaning phase to complete the training of the Naive Bayes model, and input keywords into the Naive Bayes model. The Naive Bayes model will give the phrases with the highest relevance according to the user's selection. LLM will receive these phrases and finally give a writing outline with guidance (this part is not completed, but it is already possible to communicate with LLM through HTML).
